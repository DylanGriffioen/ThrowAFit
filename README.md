


## Supported player  controllers
* Keyboard
* PlayStation Controller (tested with ps4 dualchock)


## Player in-game controls


* Move 
	* WASD [keyboard]
	* Arrows [keyboard]
	* D-pad [PS controller]
	* Left Stick [PS controller]

* Jump
	* Space [Keyboard]
	* Cross button [PS controller]
	
* Crouch (in progress)
	* C [Keyboard]
	* Square [PS controller]
	
* Pickup/Drop Object (in progress) 
	* E [Keyboard]
	* R2 [PS controller]
	
* Throw Object (in progress)
	* F [Keyboard]
	* R1 [PS controller]
	
* Pause Object (in progress)
	* Esc [Keyboard]
	* Options [PS controller]
	
	

## Add New item 

* Tag -> Item
* Layer -> ThrowableItem
	
* Collider
* Rigidbody

* Mass -> To whatever you like
	* Angular Drag -> 0.1

* ThrowableItem (script)
	* Forward Force -> To whatever you like
	* Angle -> To whatever you like
	* Knockback -> To whatever you like
	* OnHitDamage ->  To whatever you like
