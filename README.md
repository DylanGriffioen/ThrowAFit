


## Supported player controls
	- Keyboard
	- Playstation Controller (tested with ps4 dualchock)
	
## Player ingame actions
	** Move **
	- WASD [keyboard]
	- Arrows [keyboard]
	- D-pad [PS controller]
	- Left Stick [PS controller]
	** Jump **
	- Space [Keyboard]
	- Square button [PS controller]
